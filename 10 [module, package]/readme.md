## Day 10: Modules and Packages

**1. What is a Module?**

- that contains functions, classes, or variables.

- import a module into another program.

**What is a Package?**

- A folder containing multiple Python modules.
- Must have an **init**.py file to be recognized as a package.

```
mypackage/
    __init__.py
    module1.py
    module2.py

```
