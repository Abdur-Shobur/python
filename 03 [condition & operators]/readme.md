# Day 3: Conditions and Operator

| Operator | Example  | Meaning          |
| -------- | -------- | ---------------- |
| `==`     | `5 == 5` | Equal            |
| `!=`     | `5 != 3` | Not equal        |
| `>`      | `5 > 3`  | Greater than     |
| `<`      | `3 < 5`  | Less than        |
| `>=`     | `5 >= 5` | Greater or equal |
| `<=`     | `3 <= 5` | Less or equal    |

| Operator | Example          | Meaning                      |
| -------- | ---------------- | ---------------------------- |
| `and`    | `True and False` | True if both are True        |
| `or`     | `True or False`  | True if at least one is True |
| `not`    | `not True`       | Inverts the value            |
